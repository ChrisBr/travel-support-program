Overview

Transition of a state machine (a request, a reimbursement...) from one state to another. Saving the StateTransition to the database will correctly update the associated state machine automatically.

Saving StateTransition objects to the database is the right way of changing the state, as it creates a log of the change, allows the user to add textual information and notifies the change to all involved users.
